# Lucid - Node Adapter 

> The official Node adapter for Lucid CMS

## Installation

```bash
npm install @lucidcms/cloudflare-adapter
```

## lucid.config.ts/js

```typescript
import CloudflareAdapter from "@lucidcms/cloudflare-adapter"

export default lucid.config({
	adapter: CloudflareAdapter(),
    // ...other config
});
```
